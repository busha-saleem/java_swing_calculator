# Java Swing Calculator

A simple calculator built using Java Swing as a beginner GUI project.

## Features
- Basic one operator arithmetic operations with results saved for next use
- Event-driven programming using ActionListener
- Clean GUI layout

## Limitations
- No decimal input
- No keyboard support
- No advanced error handling

## Future Improvements
- Add decimal support
- Add keyboard input
- Improve UI design

## How to Run
```bash
javac MyCalculator.java
java MyCalculator
